# frontend/src/

Geplante Code-Struktur (MAUI, MVVM, Clean Architecture):
- `Core/` — Entities + Interfaces
- `Models/` — Datenstrukturen/DTOs
- `Services/` — Calorie/Timer/Profile/Session Services
- `Storage/` — SQLite/Preferences Implementierungen
- `ViewModels/` — UI Bindings + Commands
- `UI/` — Views + Komponenten (Cards, Timer, Settings)
